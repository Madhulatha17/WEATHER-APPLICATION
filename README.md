# WEATHER-APPLICATION

## Features
- Search weather by city
- Real-time weather data
- Temperature
- Humidity
- Weather conditions

## Installation

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm start
```

## Tech Stack

- MongoDB (optional)
- Express.js
- React.js
- Node.js
- OpenWeather API
